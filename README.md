## Preview

Open any HTML file in your browser to view the design. All files are self-contained with inline CSS.

## Brand Colors (v5-brand-colors)

The **v5-brand-colors** variant uses Core Interface's actual brand colors:

- **Primary:** `#b8512a` (Burnt orange/rust) - Navigation, CTAs, accents
- **Secondary:** `#2a4a5e` (Deep teal-blue) - Hero backgrounds, secondary buttons
- **Dark:** `#1a243a` (Navy) - Footer
- **Cream:** `#faf8f5` (Warm cream) - Backgrounds

## Fonts Used

- **Libre Baskerville** - Headings, serif accent
- **Nunito Sans** - Body text, UI elements

## Structure

Each design is a complete one-page website featuring:
- Fixed navigation header
- Hero section with stats
- About section
- Capabilities grid
- Contract vehicles & certifications
- Careers section
- Contact form
- Footer
